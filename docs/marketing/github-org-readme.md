# Shitware Industries

Agent-first developer tools. Open source default. Built for the way software is actually written now.

## What we build

Tools with APIs first, CLIs second, and GUIs as a convenience — never the only way in. Every tool we ship follows these rules:

- **API-first**: The API ships before the GUI.
- **CLI-native**: Every feature accessible from the terminal.
- **Open source**: Code you can read, fork, and fix. AGPL v3.
- **Budget-friendly**: Pricing for startups, not enterprises.
- **Agent-compatible**: Built for humans and agents alike.

Read the full [Agent-First Development Manifesto](https://shitware.industries/manifesto).

## Contributing

We have not shipped yet. When we do:

1. Fork the repo.
2. Open a PR. Small, focused, one thing.
3. Write a commit message that explains the "why," not the "what."
4. CI passes or you fix it.
5. We review, merge, ship.

No CLA. No contributor agreement. AGPL v3 per repo.
